## Installation
To integrate the WebRTC SDK into your XCode project add the following to your
Podfile:


```
source 'https://github.com/CocoaPods/Specs.git'
target 'YOUR_APPLICATION_TARGET_NAME_HERE' do
 platform :ios, '10.0'
 pod 'TechSeeWebRTC'
end
```

Then, run the following command

```
$ pod install
```
